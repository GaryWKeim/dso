Patch Level 1 
To install this patch
Make a copy of ~/lib/tc.jar
Unpack terracotta-ee-3.4.0.tar.gz 
Copy tc.jar to the /lib directory 
Remove the existing  terracotta-toolkit-runtime* from the /common directory
Copy terracotta-toolkit-runtime.2.0.0-ee-patch*.jar to the /common directory
Make sure that classpath no longer refers to the old runtime jar
